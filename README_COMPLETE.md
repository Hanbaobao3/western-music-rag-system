# 西方音乐史RAG问答系统 - 完整版

## 🎉 系统概述

这是一个功能完整的RAG（检索增强生成）问答系统，专为西方音乐史知识库设计，集成以下功能：

### 🛠️ 技术特点
- **前端**：中世纪羊皮纸风格的现代化网页界面
- **后端**：Python Flask服务器，提供RESTful API
- **RAG引擎**：向量检索 + AI增强答案生成
- **AI模型**：Claude 3.5 Sonnet（通过Z.ai平台）

## 🎨 设计特点

### 中世纪羊皮纸风格
- **背景**：经典羊皮纸纹理效果
- **色彩**：
  - 深黑色 (#1a1a1a)：主要文字和UI元素
  - 香槟金 (#f7e7ce)：强调色和按钮
  - 大理石白 (#fdfbf7)：背景和卡片
  - 金色强调 (#c9a227)：图标和特殊元素
- **玻璃态效果**：半透明背景 + 毛玻璃效果

### 字体设计
- **标题字体**：Playfair Display（衬线体）
  - 体现历史感和权威性
  - 用于标题和重要标签
- **正文字体**：Inter（无衬线体）
  - 保证阅读清晰度
  - 用于正文和用户输入

### 动画效果
- 平滑的过渡动画（0.3s cubic-bezier）
- 消息淡入效果
- 卡片悬停提升效果
- 通知滑入动画

## 🔧 配置说明

### .env 文件配置
```bash
# Z.ai Platform API Configuration
ZAI_API_KEY=your_api_key_here
ZAI_API_BASE=https://api.z.ai/api/anthropic

# Model Configuration
ANTHROPIC_MODEL=claude-3-5-sonnet-20241022

# Server Configuration
HOST=0.0.0.0
PORT=5000
DEBUG=False
```

### 系统配置
- 向量存储路径：自动处理西方音乐史PDF
- 检索参数：TopK=5，最小相似度=0.7
- 切块参数：512 tokens per chunk，50 tokens overlap

## 🔌 API接口

### 1. 查询接口
**端点**: `POST /api/query`

**请求体**:
```json
{
  "query": "巴洛克音乐有什么特点？",
  "period": "baroque",  // 可选：medieval, renaissance, baroque, classical, romantic, modern, all
  "use_ai": true      // 是否使用AI增强答案
}
```

**响应**:
```json
{
  "success": true,
  "answer": "根据参考文档...",
  "retrieved_docs": [...],
  "period": "baroque",
  "ai_generated": true,
  "model": "claude-3-5-sonnet-20241022",
  "timestamp": "2026-04-29T14:30:00"
```

## 🔍 故障排查

### 常见问题

**Q: 网页无法连接到服务器**
A: 
1. 检查服务器是否正在运行：http://localhost:5000/api/health
2. 确认.env文件中的API密钥正确
3. 检查端口5000是否被占用

**Q: API返回错误**
A:
1. 检查网络连接
2. 验证ZAI_API_KEY是否有效
3. 查看服务器日志获取详细错误信息

**Q: 检索结果不准确**
A:
1. 确认PDF已正确处理：运行离线处理流程
2. 检查向量存储文件是否存在
3. 尝试调整时期选择以聚焦查询范围

**Q: 启动脚本出现乱码**
A: 已修复编码问题，确保系统区域设置正确
