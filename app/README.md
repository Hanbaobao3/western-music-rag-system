## 📚 知识库内容

### 支持的时期

1. **古代与中世纪音乐** (公元前～1400年)
   - 古希腊音乐理论：调式体系、数学理论、伦理功能
   - 基督教音乐发展：早期基督教、弥撒曲结构、教会调式
   - 中世纪复调音乐：游吟诗人、记谱法发展、圣母院乐派
   - 文艺复兴变革：世俗音乐与宗教音乐并驾齐驱

2. **文艺复兴音乐** (约1400—1600年)
   - 早期文艺复兴：勃艮第乐派、法-佛兰德斯乐派
   - 宗教改革与音乐：马丁·路德、德语圣咏、特伦托大公会议
   - 世俗声乐体裁：英国牧歌、法国尚松、德国利德
   - 器乐独立发展：托卡塔、里切尔卡尔、变奏曲

3. **巴洛克音乐** (约1600—1750年)
   - 歌剧的诞生与发展：佛罗伦萨卡梅拉塔、克劳迪奥·蒙泰韦尔迪
   - 法国巴洛克音乐：让-巴蒂斯特·吕利、让-菲利普·拉莫
   - 英国巴洛克音乐：亨利·珀塞尔
   - 巴洛克器乐体裁：安东尼奥·维瓦尔第、阿尔坎杰洛·科雷利

4. **古典主义音乐** (约1750—1820年)
   - 风格前奏：华美风格、敏感风格、曼海姆乐派
   - 古典主义重要曲式：回旋曲式、变奏曲式
   - 主要特征：理性与秩序、简洁明晰、奏鸣曲式确立
   - 代表人物：海顿、莫扎特、贝多芬

5. **浪漫主义音乐** (约1820—1900年)
   - 浪漫主义音乐特点：情感至上、标题音乐、民族主义兴起
   - 早期浪漫主义：韦伯、门德尔松、舒曼
   - 浪漫主义中期：约翰内斯·勃拉姆斯
   - 浪漫主义歌剧：大歌剧、抒情歌剧、歌剧喜歌剧
   - 晚期浪漫主义：古斯塔夫·马勒、理查·施特劳斯

6. **20世纪音乐** (约1900—2000年)
   - 印象主义：莫里斯·拉威尔、德彪西
   - 表现主义与十二音体系：维也纳乐派、勋伯格、贝尔格、韦伯恩
   - 新古典主义与民族主义：普罗科菲耶夫、艾夫斯
   - 战后先锋派：整体序列主义、偶然音乐、电子音乐
   - 简约主义与梅西安：史蒂夫·赖希、菲利普·格拉斯、奥利维埃·梅西安

7. **核心名词解释**
   - 音乐理论概念：格里高利圣咏、奥尔加农、经文歌、等节奏
   - 巴洛克概念：通奏低音、情感理论、利都奈罗、达卡波咏叹调
   - 古典主义概念：主导动机、奏鸣曲式、整体艺术作品、交响诗
   - 艺术歌曲：浪漫主义声乐体裁
   - 20世纪概念：十二音技法、说话-唱法、具体音乐、偶然音乐、简约主义

8. **考研记忆口诀**
   - 时期口诀：中世纪、文艺复兴、巴洛克、古典主义、浪漫主义、20世纪
   - 便于记忆：按时期和重要人物组织
   - 考试重点：涵盖主要作曲家、作品、风格特征

## 📋 答题规范

### 名词解释
**识别关键词**: 什么是、定义、名词解释、解释、概念

**答题结构**:
- **核心概念**: 直接给出定义和要点
- **历史意义**: 在音乐史上的地位和影响
- **相关内容**: 与该概念相关的其他要素

### 简答题
**识别关键词**: 特点、特征、风格、主要、简述、简答

**答题结构**:
- **主要特征**: 列出3-5个重要特征
- **代表人物**: 该时期的主要作曲家
- **历史意义**: 对后世音乐发展的影响

### 论述题
**识别关键词**: 论述、分析、发展、演变、过程、历史、影响、意义、贡献

**答题结构**:
- **一、背景**: 时期发展的历史背景
- **二、主要内容**: 前4个重要特征或发展
- **三、风格特征**: 后续重要特征
- **四、代表人物**: 主要作曲家和作品
- **五、历史意义**: 对音乐发展的重要影响

### 记忆查询
**识别关键词**: 记忆、口诀、考研、总结、要点

**答题结构**:
- **时期口诀**: 按时期组织的记忆要点
- **人物作品**: 主要作曲家和代表作品
- **风格特征**: 各个时期的核心特征
- **历史意义**: 各个时期对音乐史发展的贡献
